Git, Github
======================
## 1. Git
* git status 변경사항 확인
* git add 파일 추가
* git add . 모든 파일 추가

git commit
-------
|작업|VI 명령어|상세|
|------|---|---|
|입력 시작|i|텍스트 입력 모드로 전환|
|입력 종료|ESC|명령어 입력 모드로 전환|
|저장 없이 종료|:q||
|저장 없이 강제 종료|:q!|입력한 것 있을 때 사용|
|저장하고 종료|:wq|압력한 것 있을 때 사용|
|위로 스크롤|k|git log등에서 내역이 길 때 사용|
|아래로 스크롤|j|git log등에서 내역이 길 때 사용|
